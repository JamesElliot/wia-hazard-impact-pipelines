# Example notebooks

The notebooks in `examples/` demonstrate and visually inspect the supported
Python pipelines. They are not the source of truth: reusable implementation
belongs in `src/wia_pipelines/` and supported command-line entry points belong
under `wia-hazards`.

Saved cell outputs and execution counts are cleared before publication. Run a
notebook only after creating the documented environment, installing the
package in editable mode, and supplying the required external data.
